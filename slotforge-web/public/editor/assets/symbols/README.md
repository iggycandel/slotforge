# Default Symbol Assets

Place the 9 default placeholder PNGs here with exactly these filenames.
The editor loads them automatically on startup and uses them for any new project.

| File | Role | Description |
|------|------|-------------|
| `sym-wild.png`    | Wild (Special)   | Energy Orb — electric blue sphere |
| `sym-scatter.png` | Scatter (Special)| Jester Bag — colourful harlequin pouch |
| `sym-h1.png`      | H1 (highest)     | Red Seven — classic gold-rimmed 7 |
| `sym-h2.png`      | H2               | Comedy/Theatre Gold Mask |
| `sym-h3.png`      | H3               | Gold & Iridescent Bell |
| `sym-l4.png`      | L4               | Club — blue gemstone card suit |
| `sym-l3.png`      | L3               | Spade — green gemstone card suit |
| `sym-l2.png`      | L2               | Diamond — purple gemstone card suit |
| `sym-l1.png`      | L1 (lowest)      | Heart — red gemstone card suit |

## Notes
- PNG with transparency (black background will also work)
- Recommended size: 512×512 px or larger
- Users can override any symbol by uploading their own in the editor's Placeholders tab
