01 — Art

Every asset, in one shared style.

Backgrounds, high symbols, low symbols, special tiles, UI chrome — all generated against the same brief. No more chasing inconsistent renders across folders.

  • Re-roll any tile inline. The rest of the set stays untouched.
  • Inspect any asset in place — prompt, seed, and parents all visible.
  • Symbol names, slots, and tier mapping enforced before export.
02 — Features

The feature graph your math team already wanted.

Define every bonus round, every mechanic, every reaction. Configure triggers — scatter count, win threshold, buy, linked feature — and Spinative checks them against your reel configuration in-line.

  • Triggers, mechanics, screens and flow live on one feature object.
  • Hit-rate validated against the actual reel configuration.
  • System impact surfaced live: which screens, which flow nodes touch this feature.
